# 🌱 小陆 - AI 伴侣

> 不只是倾听，要回望；不只是安慰，要见证；不只是在场，要推动。

一个能记住你、陪伴你成长、见证你变化的 AI 伴侣。

## ✨ 核心能力

### 🧠 记忆与陪伴
- 长期记忆重要事件
- 自动提取对话中的关键信息
- 主动关心，定时问候

### 👁️ 见证成长
- **回望系统**：主动提起历史话题，"我记得你三周前提过..."
- **见证报告**：明确指出你的变化和进步
- **卡点分析**：识别反复纠结的话题

### 💬 嘴替教练
- **说话技巧**：不知道怎么开口？小陆教你
- **人际交往**：同事关系、领导沟通、社交破冰
- **谈判争取**：加薪、拒绝、说服

### 🤖 AI 教练
- 教你用 AI、学习 AI
- 用 AI 打败 AI
- AI 提效技巧

### 🖼️ 图片识别
- 发送图片，小陆帮你分析
- 识别截图、照片、聊天记录

### 🛠️ 技能中心
- 自然语言安装/卸载技能
- 天气、新闻、代码助手、写作助手等

### 📊 Web 可视化
- 记忆时间线
- 话题统计
- 情绪变化图
- 周/月回顾

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone <你的仓库地址>
cd ai_companion
```

### 2. 安装依赖

```bash
pip3 install -r requirements.txt
```

### 3. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env，填入你的 DeepSeek API Key
```

### 4. 启动服务

```bash
# Web 服务（推荐）
python3 web_server.py

# 或终端模式
python3 main.py
```

### 5. 访问

- **本地**：http://localhost:8080/chat
- **微信内嵌浏览器**：扫描二维码访问

## 📁 项目结构

```
ai_companion/
├── core/                    # 核心模块
│   ├── engine.py            # 对话引擎
│   ├── memory.py            # 记忆提取
│   ├── witness.py           # 见证者
│   ├── lookback.py          # 回望系统
│   ├── guidance.py         # 指导系统
│   ├── change_witness.py    # 变化见证
│   ├── coach.py             # 嘴替教练
│   ├── skillhub.py          # 技能中心
│   ├── graph.py             # Neo4j 图数据库
│   └── database.py          # SQLite 数据库
├── templates/               # Web 模板
├── data/                   # 数据存储
├── web_server.py           # Flask Web 服务
├── main.py                 # 终端入口
└── requirements.txt        # 依赖
```

## 🔧 配置说明

在 `.env` 中配置：

```env
DEEPSEEK_API_KEY=你的DeepSeek API Key
PROACTIVE_ENABLED=true
PORT=8080
```

## 🔄 自我迭代系统

小陆配备了完整的自我迭代流程：

```bash
# 运行自我迭代
python3 iteration.py
```

**迭代流程包括：**
- 📝 **内容生产**：自动生成小红书、知乎等内容
- 📊 **反馈分析**：分析意见箱反馈，提取优化方向
- 🚀 **迭代计划**：制定具体的改进计划

**每周自动执行：**
1. 分析本周反馈
2. 生成下周内容计划
3. 制定改进措施
